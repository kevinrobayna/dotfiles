### Dependencies

* Vim
* Git

### Mac OS X

**Homebrew:** [http://brew.sh/](http://brew.sh/)

    $ brew install vim git

### How to install

    $ git clone https://github.com/kevinrobayna/dotfiles.git

    $ cd dotfiles/

    $ make deps #Optional

    $ make mac 

### How to Remove Backup files
    
    $ make rm-backup
